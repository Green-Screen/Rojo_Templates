# Installation
# Current version: X.X.XX

## Method #1 - Roblox
Download straight from the [roblox marketplace](https://create.roblox.com/store/asset) this will only contain the module itself.

## Method #2 - Github
Download straight from the releases subdirectory of this [repository](https://github.com/Green-Screen)

### Download options

    1. Install the Release.zip file for just the runtime module
    2. Install the ReleaseWithTests.zip file for runtime and test scripts

## Method #3 - Git Clone
Clone this repository straight into your project with Git

### Git steps:

```bash
git clone https://github.com/Green-Screen
```
