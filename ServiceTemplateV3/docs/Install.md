# Installation
# Current version: X.X.XX

## Method #1 - Roblox
Download straight from the [roblox marketplace](https://create.roblox.com/store/asset) this will only contain the module itself.

## Method #2 - Github
Download straight from the releases subdirectory of this [repository](https://github.com/Green-Screen)

### Download options

    1. Install the [CLASS.VX.X.XXWithDependencies.zip](https://ExampleDownloadLink) file for just the runtime module with its dependencies
    2. Install the [CLASS.VX.X.XXWithTests.zip](https://ExampleDownloadLink) file for runtime and test scripts
    3. Install the [CLASS.VX.X.XX.zip](https://ExampleDownloadLink) file WITHOUT dependencies. **[Dependencies](#dependencies) are needed to run**

## Method #3 - Git Clone
Clone this repository straight into your project with Git

### Git steps:

```bash
git clone https://github.com/Green-Screen/SecureClasses.git
```
